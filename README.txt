Chương trình từ điển bằng Java do nhóm ABC thực hiện.
Thành viên nhóm: Hoàng Quốc Cường (20020255), Dương Hùng Anh (20020254), Nguyễn Xuân Bách (20020097)
- Lập trình bằng Java, thiết lập giao diện đồ họa cơ bản bằng JavaFX và giao diện cải tiến bằng CSS
- Sử dụng Google Translate API để dịch đa ngôn ngữ và Java TTS API để phát âm
- Tích hợp dữ liệu từ điển từ Online Dictionary API bao gồm từ ngữ, cách phát âm, dạng từ, ví dụ và từ đồng nghĩa

Giới thiệu giao diện:
- File: đóng chương trình
- Help:
 + How to use: hướng dẫn cơ bản cách sử dụng chương trình
 + About Us: giới thiệu thông tin thành viên nhóm
- Từ điển được chia làm 2 phần: bên trái là danh sách từ, bên phải là thanh ghi và giải nghĩa từ
- Nút Speaker (Loa): phát âm từ/ câu được viết trong thanh ghi
- Nút Google Translate: dịch nội dung trong thanh ghi sang ngôn ngữ được chọn bên dưới
- Nút Add a word: thêm một từ vào danh sách
- Nút Delete a word: xóa một từ trong danh sách
